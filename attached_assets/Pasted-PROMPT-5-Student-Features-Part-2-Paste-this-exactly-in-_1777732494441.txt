STEP 5 — BATTLE MODE 1V1:
Build a real-time 1v1 quiz battle system:
How it works:

Student goes to Battle Mode section
Clicks "Find Opponent" → matched with another online student randomly
OR clicks "Challenge Friend" → generates unique battle link → share via WhatsApp
Friend opens link → joins battle
Both students get SAME 10 questions simultaneously
60 second timer per question
Both see each other's progress live (answered/not answered — no peeking at answers)
After all 10 questions → results shown side by side
Winner gets +100 XP, loser gets +20 XP (participation)
Battle history saved in MongoDB
Win/loss record shown on profile

Technical implementation:

Use Socket.io for real-time connection (already likely in stack or add it)
Battle room created in MongoDB with unique ID
Both players connected to same room
Questions synced — both get identical question set
If opponent disconnects → show "Opponent left" → current student wins automatically
Maximum wait time for opponent: 60 seconds → then show "No opponent found, try again"

UI:

Split screen on desktop (you vs opponent)
Stacked on mobile (your score on top, opponent below)
Live score updating every second
Opponent name shown as username only (no personal details)


STEP 6 — CHALLENGE A FRIEND:

After friend completes → both see comparison result
Who scored more, who was faster, head to head breakdown
Challenge expires after 48 hours
Challenge history saved on profile


STEP 7 — RAPID FIRE MODE:

+30 XP for completing rapid fire
+50 XP bonus if all 10 correct
Mobile friendly — big option buttons, easy to tap fast


STEP 8 — COMEBACK STREAK BONUS:

System tracks last activity date per student in MongoDB
If student was on streak and missed 1-3 days:

Show motivational message

Streak resets to 0
Still show welcome back message
Award +20 XP "fresh start" bonus
Show: "Your streak reset but your XP and progress are safe!"


Never punish student harshly — always encouraging tone
